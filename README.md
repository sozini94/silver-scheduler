# 실버 프로그램 주간 수업표 생성기

Next.js로 만든 주간보호센터용 프로그램 시간표 자동 생성 예제입니다.

## 포함 기능

- 255개 실버 프로그램 데이터 포함
- 운영 요일 선택: 월~금, 월~토, 월~일
- 하루 수업 횟수 선택: 1~6회
- 프로그램 종류 다중 선택: 인지, 신체, 정서, 사회성, 생활, 특별행사
- 선택 조건에 맞는 주간 수업표 자동 생성
- 오전에는 인지 프로그램 우선, 신체 프로그램 하루 1회 우선 배치 옵션
- 프로그램별 시간, 난이도, 이동성, 준비물, 설명 표시

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속하세요.

## 주요 파일

- `src/data/programs.ts`: 255개 프로그램 데이터
- `src/lib/scheduler.ts`: 시간표 추천 로직
- `src/app/page.tsx`: 화면 UI
- `src/app/globals.css`: 스타일
